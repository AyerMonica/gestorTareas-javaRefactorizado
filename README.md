# Gestor de Tareas refactorizado en Java

Aplicación de consola desarrollada como proyecto de práctica del ciclo
Desarrollo de Aplicaciones Multiplataforma (DAM).

## Funcionalidades
- Añadir tareas
- Mostrar la lista de tareas
- Eliminar tareas
- Salir de la aplicación

## Tecnologías utilizadas
- Java
- ArrayList
- Scanner
- Programación estructurada

## Descripción
La aplicación muestra un menú por consola que permite al usuario gestionar
una lista de tareas introducidas por teclado. Se utilizan estructuras básicas
de control, colecciones y entrada/salida estándar. El código está refactorizado
en métodos separados para mejorar la legibilidad y modularidad.

## Objetivo del proyecto
Afianzar los conocimientos básicos de Java y la lógica de programación mediante
un proyecto práctico.
